# 快速开始指南 - Hugo + Cloudflare Pages

## 🚀 5分钟快速搭建

### 前置要求
- ✅ macOS系统
- ✅ 已安装Homebrew（如未安装：`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`）
- ✅ 已安装Git
- ✅ 拥有GitHub/GitLab账户
- ✅ Cloudflare账户和域名

---

## 步骤1：安装Hugo

```bash
brew install hugo
hugo version  # 验证安装
```

---

## 步骤2：创建博客项目

```bash
# 在项目目录下执行
hugo new site . --force

# 初始化Git
git init
```

---

## 步骤3：安装主题（推荐PaperMod）

```bash
# 添加主题
git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

# 或者直接克隆
git clone https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod
```

---

## 步骤4：创建配置文件

创建 `config.yaml`：

```yaml
baseURL: 'https://yourdomain.com'  # 替换为你的域名
languageCode: 'zh-cn'
title: '我的博客'
theme: PaperMod

params:
  env: production
  title: '我的博客'
  description: '个人技术博客'
  
  homeInfoParams:
    Title: "欢迎来到我的博客"
    Content: "分享技术、生活与思考"
  
  socialIcons:
    - name: github
      url: "https://github.com/yourusername"
    - name: email
      url: "mailto:your@email.com"

menu:
  main:
    - identifier: home
      name: 首页
      url: /
      weight: 10
    - identifier: archives
      name: 归档
      url: /archives/
      weight: 20
    - identifier: search
      name: 搜索
      url: /search/
      weight: 30
```

---

## 步骤5：创建第一篇文章

```bash
hugo new posts/hello-world.md
```

编辑 `content/posts/hello-world.md`：

```markdown
---
title: "Hello World"
date: 2024-01-01T10:00:00+08:00
draft: false
tags: ["博客", "开始"]
---

# 欢迎来到我的博客！

这是我的第一篇文章。

## 功能特性

- ⚡ 极快的构建速度
- 🎨 美观的主题
- 📱 响应式设计
- 🔍 SEO优化

开始写作吧！
```


## 步骤8：推送到Git仓库

```bash
# 创建GitHub仓库（在GitHub网站上）

# 添加远程仓库
git remote add origin https://github.com/yourusername/my-blog.git

# 提交代码
git add .
git commit -m "Initial commit: Hugo blog setup"
git branch -M main
git push -u origin main
```

---

## 步骤9：部署到Cloudflare Pages

1. **登录Cloudflare Dashboard**
   - 访问：https://dash.cloudflare.com/

2. **进入Pages**
   - 左侧菜单 → Pages
   - 点击 "Create a project"

3. **连接Git仓库**
   - 选择 "Connect to Git"
   - 授权GitHub/GitLab
   - 选择你的仓库

4. **配置构建设置**
   ```
   Framework preset: Hugo
   Build command: hugo --minify
   Build output directory: public
   ```

5. **环境变量（可选）**
   - 如果需要，可以添加环境变量
   - 例如：`HUGO_VERSION=0.120.0`

6. **保存并部署**
   - 点击 "Save and Deploy"
   - 等待构建完成（通常1-2分钟）

---

## 步骤10：配置自定义域名

1. **在Cloudflare Pages项目中**
   - 进入 "Custom domains"
   - 点击 "Set up a custom domain"

2. **添加域名**
   - 输入你的域名（如：blog.yourdomain.com）
   - Cloudflare会自动配置DNS和SSL证书

3. **等待生效**
   - DNS配置通常几分钟内生效
   - SSL证书自动配置（可能需要几分钟）

---

## 步骤11：更新baseURL

部署成功后，更新 `config.yaml` 中的 `baseURL`：

```yaml
baseURL: 'https://blog.yourdomain.com'  # 使用实际域名
```

然后提交并推送：

```bash
git add config.yaml
git commit -m "Update baseURL"
git push
```

## 常见问题

### Q: 构建失败？
- 检查 `config.yaml` 语法
- 确认主题已正确安装
- 查看Cloudflare Pages构建日志

### Q: 样式丢失？
- 确认主题路径正确
- 检查 `config.yaml` 中的 `theme: PaperMod`

### Q: 文章不显示？
- 确认文章 `draft: false`
- 检查文件路径：`content/posts/`

### Q: 域名无法访问？
- 检查DNS配置
- 等待SSL证书生效（最多24小时）
- 确认Cloudflare Pages域名配置正确
