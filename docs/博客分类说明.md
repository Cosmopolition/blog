# 博客分类说明

本文档定义了博客的分类体系，用于组织和管理文章内容。

## 📂 分类结构

### 1. 技术

技术类文章，涵盖开发相关的所有内容。

- **flutter** - Flutter开发相关文章
- **编程语言** - 各种编程语言的学习和使用
- **开发随笔/踩坑记** - 开发过程中的经验总结、问题解决、踩坑记录

### 2. 生活

生活类文章，记录日常生活和体验。

- **旅行与体验** - 旅行游记、生活体验、见闻分享
- **运动与健康** - 运动记录、健康管理、健身心得

### 3. 创作/作品

创作和作品展示类文章。

- **ios应用** - iOS应用开发、发布、使用体验
- **摄影** - 摄影作品、摄影技巧、器材分享

### 4. 学习路径

学习相关的规划和资源整理。

- **课程/书单/资源整理** - 学习计划、推荐书单、优质资源汇总

### 5. 日志

定期总结和回顾类文章。

- **月记** - 月度总结和回顾
- **年度回顾** - 年度总结和回顾

## 📝 使用说明

### 在文章中使用分类

在文章的 front matter 中使用 `categories` 字段：

```yaml
---
title: "文章标题"
date: 2025-12-20T10:00:00+08:00
draft: false
tags: ["标签1", "标签2"]
categories: ["技术", "flutter"]  # 可以设置多个分类
description: "文章描述"
---
```

### 分类选择建议

- 每篇文章建议选择 **1-2个分类**，最多不超过3个
- 优先选择最相关的分类
- 分类用于内容组织，标签用于更细粒度的标记

### 分类与标签的区别

- **分类（categories）**：用于内容的大类组织，相对固定
- **标签（tags）**：用于细粒度标记，可以更灵活，如具体技术名称、地点、人名等

## 🔄 分类更新

如需添加或修改分类，请更新本文档并同步更新文章模板。
