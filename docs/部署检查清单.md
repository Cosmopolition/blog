## 📋 待执行步骤

### 1. 安装Hugo

```bash
brew install hugo
hugo version  # 验证安装
```

### 2. 安装PaperMod主题

```bash
# 运行安装脚本
./install-theme.sh

# 或手动安装
git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod
```

### 3. 修改配置

编辑 `config.yaml`：
- 修改 `baseURL` 为你的实际域名
- 修改 `title` 和 `description`
- 更新 `socialIcons` 中的链接

### 4. 本地测试

```bash
# 启动本地服务器
hugo server -D

# 访问 http://localhost:1313 查看效果
```

### 5. 推送到Git仓库

```bash
# 创建GitHub/GitLab仓库后执行
git add .
git commit -m "Initial commit: Hugo blog setup"
git branch -M main
git remote add origin https://github.com/yourusername/my-blog.git
git push -u origin main
```

### 6. 部署到Cloudflare Pages

参考 `cloudflare-pages.md` 文件中的详细步骤。

## 🐛 常见问题

### Hugo未安装
```bash
brew install hugo
```

### 主题安装失败
- 检查网络连接
- 尝试手动下载主题到 `themes/PaperMod` 目录
- 或使用其他主题

### 本地预览失败
- 确认Hugo已安装
- 检查 `config.yaml` 语法
- 确认主题路径正确

### 构建失败
- 查看Cloudflare Pages构建日志
- 检查配置文件语法
- 确认所有依赖已提交到仓库
