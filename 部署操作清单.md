# ✅ 部署操作清单

使用此清单逐步完成部署，确保不遗漏任何步骤。

## 📋 部署前准备

- [ ] 确认Hugo已安装：`hugo version`
- [ ] 确认代码已准备就绪：`git status`
- [ ] 确认`config.yaml`中的`baseURL`已配置为：`https://mingxinwang.com`
- [ ] 准备好GitHub账户
- [ ] 准备好Cloudflare账户
- [ ] 确认域名已在Cloudflare DNS中

## 第一部分：Git提交

### 步骤1：创建GitHub仓库

- [ ] 访问 https://github.com/new
- [ ] 创建新仓库（名称：`my-blog` 或自定义）
- [ ] **不要**勾选"Initialize with README"
- [ ] 点击"Create repository"
- [ ] 复制仓库URL

### 步骤2：提交代码

```bash
# 执行以下命令
git add .
git commit -m "Initial commit: Hugo blog"
git remote add origin https://github.com/yourusername/my-blog.git
git branch -M main
git push -u origin main
```

- [ ] 执行 `git add .`
- [ ] 执行 `git commit`
- [ ] 执行 `git remote add origin`（替换为你的仓库URL）
- [ ] 执行 `git push`
- [ ] 在GitHub上确认文件已上传

## 第二部分：Cloudflare Pages部署

### 步骤3：创建Pages项目

- [ ] 登录 https://dash.cloudflare.com/
- [ ] 进入 **Workers & Pages** → **Pages**
- [ ] 点击 **"Create a project"**
- [ ] 选择 **"Connect to Git"**
- [ ] 授权GitHub/GitLab
- [ ] 选择你的仓库：`my-blog`
- [ ] 点击 **"Begin setup"**

### 步骤4：配置构建设置

- [ ] Project name: `my-blog`
- [ ] Production branch: `main`
- [ ] Framework preset: `Hugo`
- [ ] Build command: `hugo --minify`
- [ ] Build output directory: `public`
- [ ] Root directory: `/` (留空)

### 步骤5：添加环境变量

- [ ] 点击 **"Add environment variable"**
- [ ] Variable name: `HUGO_VERSION`
- [ ] Value: `0.147.4`
- [ ] 点击 **"Add"**

### 步骤6：保存并部署

- [ ] 点击 **"Save and Deploy"**
- [ ] 等待构建完成（1-3分钟）
- [ ] 确认构建成功（绿色✓）
- [ ] 记录预览URL：`https://my-blog.pages.dev`

## 第三部分：配置域名

### 步骤7：配置自定义域名

- [ ] 在Pages项目中点击 **"Custom domains"**
- [ ] 点击 **"Set up a custom domain"**
- [ ] 输入域名：`mingxinwang.com`
- [ ] 点击 **"Continue"**
- [ ] 确认DNS记录已自动创建

### 步骤8：验证域名配置

- [ ] 在Cloudflare DNS中确认CNAME记录存在
- [ ] 确认SSL证书状态为"Active"（可能需要等待5-15分钟）
- [ ] 访问 `https://mingxinwang.com` 测试
- [ ] 确认网站正常显示
- [ ] 确认HTTPS正常工作（浏览器显示锁图标）

## 第四部分：验证和测试

### 步骤9：功能测试

- [ ] 首页正常显示
- [ ] 导航菜单正常
- [ ] 文章页面正常
- [ ] 移动端显示正常
- [ ] 样式正常加载

### 步骤10：自动部署测试

- [ ] 修改一个文件（如README.md）
- [ ] 提交并推送：`git add . && git commit -m "Test" && git push`
- [ ] 在Cloudflare Pages中确认自动触发部署
- [ ] 等待部署完成
- [ ] 确认网站已更新

## ✅ 完成检查

- [ ] 所有步骤已完成
- [ ] 网站可以正常访问
- [ ] 自动部署正常工作
- [ ] 域名配置正确
- [ ] SSL证书已激活
