# ⚡ 快速部署指南 - 5分钟部署

这是简化版的部署指南，适合快速操作。详细步骤请参考 [完整部署流程.md](./完整部署流程.md)。

## 🚀 三步部署

### 第一步：提交代码到GitHub

```bash
# 1. 添加所有文件
git add .

# 2. 提交
git commit -m "Initial commit: Hugo blog"

# 3. 添加远程仓库（替换为你的GitHub信息）
git remote add origin https://github.com/yourusername/my-blog.git
git branch -M main
git push -u origin main
```

**如果没有GitHub仓库：**
1. 访问 https://github.com/new
2. 创建新仓库（不要初始化README）
3. 复制仓库URL，替换上面的命令

### 第二步：在Cloudflare Pages中部署

1. **登录Cloudflare Dashboard**
   - 访问：https://dash.cloudflare.com/
   - 进入 **Workers & Pages** → **Pages**

2. **创建项目**
   - 点击 **"Create a project"**
   - 选择 **"Connect to Git"**
   - 授权GitHub并选择你的仓库

3. **配置构建设置**
   ```
   Framework preset: Hugo
   Build command: hugo --minify
   Build output directory: public
   ```

4. **添加环境变量（推荐）**
   ```
   Variable name: HUGO_VERSION
   Value: 0.147.4
   ```

5. **保存并部署**
   - 点击 **"Save and Deploy"**
   - 等待1-3分钟

### 第三步：配置域名

1. **在Pages项目中**
   - 进入 **"Custom domains"**
   - 点击 **"Set up a custom domain"**
   - 输入：`mingxinwang.com`

2. **等待生效**
   - DNS配置：1-5分钟
   - SSL证书：5-15分钟

3. **验证**
   - 访问 `https://mingxinwang.com`
   - 确认网站正常显示

## ✅ 完成！

现在你的博客已经上线了！

## 🔄 后续更新

每次更新内容后：

```bash
git add .
git commit -m "Update content"
git push
```

Cloudflare Pages会自动部署更新。
